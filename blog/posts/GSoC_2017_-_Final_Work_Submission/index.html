GSoC 2017 - Final Work Submission

Posted on 28/08/2017 at 08:00 AM by The Vibe.

Software Development Documentation Ruby GSoC 2017

Through this blog post, I'd like to comprehensively document all of my work done through the entire Google Summer of Code 2017. This involves links to all code documentations, blog posts, repositories, discussions and code merges. A huge token of thanks for all the support extended by mentors Victor Shepelev, Sameer Deshmukh, Lokesh Sharma and the entire Ruby Science Foundation community for their swift responses, open interactions and healthy discussions.


Connected repositories

  • Daru : A Data Analysis framework in RUby.
  • Daru-IO : A plugin gem to daru, that extends Import-Export features.
  • Daru-View : A plugin gem to daru, that extends visualization features.
  • Sample Rails App : A sample rails app, showcasing the easeness introduced by the combination of daru, daru-io and daru-view with Rails.
